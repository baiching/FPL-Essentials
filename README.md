## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/baiching/FPL-Essentials.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd fpl-essentials-api
   ```
3. **Run the application:**
   ```bash
   mvn spring-boot:run
   ```
